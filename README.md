# 项目名称：ADHD 沉浸式正念装置
### 项目简介
本项目是一个针对 ADHD（注意力缺陷及多动障碍）群体的沉浸式生理反馈系统。通过感官石捕获心率信号，实时驱动数字星云律动，引导用户从深呼吸进入深度觉察状态。
### 核心功能
- **硬件同频**：支持 Arduino 心率传感器实时接入。
- **视觉干预**：基于非牛顿流体算法的低熵视觉设计。
- **正向激励**：提供 12%-46% 的平静度提升反馈
### 如何使用
1. 使用 Chrome 浏览器打开 [点击查看预览链接]
2. 确保 Arduino 已连接并上传了配套的心率采集代码。
3. 点击右上角“连接感官石”即可开始。
